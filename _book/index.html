简介

什么是 Docker

  • 开源项目
  • 基于 Go 语言实现
  • 遵从 Apache 2.0 协议
  • 以 Linux 容器（LXC）等技术为基础
    • 在 LXC 的基础上进行了进一步封装
    • 用户不用关心容器管理
  • 目标：实现轻量级的操作系统虚拟化解决方案

为什么使用 Docker

优势

  • 秒级启动
  • 系统资源利用率高，一台主机上可以同时运行数千个 Docker 容器
  • 可以在任意平台上运行
  • 部署简单，快速交付
